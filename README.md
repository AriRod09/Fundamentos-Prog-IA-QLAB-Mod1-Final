# Fundamentos-Prog-IA-QLAB-Mod1-Final
Repositorio del trabajo final (Módulo 1) de Katia Villalobos, Diego Canales y Ariana Rodriguez para el curso de Fundamentos de Programación para IA Generativa en Ciencias Sociales y Gestión Pública (2026-01) en QLAB-PUCP. Se analiza la relación entre nivel educativo, IDH y gasto de bolsillo en salud en Perú mediante el procesamiento de la ENAHO.
