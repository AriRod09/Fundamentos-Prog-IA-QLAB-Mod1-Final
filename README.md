# Gasto de Bolsillo en Salud en Perú (2020-2024)

Repositorio del trabajo final (Módulo 1) de Katia Villalobos, Diego Canales 
y Ariana Rodriguez para el curso de Fundamentos de Programación para IA 
Generativa en Ciencias Sociales y Gestión Pública (2026-01) en QLAB-PUCP.

Se analiza la relación entre nivel educativo, IDH provincial y gasto de 
bolsillo en salud en Perú (2020-2024) mediante el procesamiento de la 
Encuesta Nacional de Hogares (ENAHO) y datos del Informe de Desarrollo 
Humano 2025 del PNUD Perú.

## Estructura del repositorio

| Archivo | Descripción |
|---------|-------------|
| `0_MASTER_SCRIPT.ipynb` | Script principal - ejecuta todos los notebooks secuencialmente |
| `1_INTRODUCCION.md` | Introducción y pregunta de investigación |
| `2_REVISION_LITERATURA.md` | Revisión de literatura |
| `3_HERRAMIENTAS.md` | Elementos de programación y herramientas de IA Generativa |
| `8_BIBLIOGRAFÍA.md` | Referencias bibliográficas |

| `scripts/4_CARGA_LIMPIEZA.ipynb` | Carga, limpieza y construcción de variables |
| `scripts/5_DESCRIPTIVAS.ipynb` | Estadística descriptiva |
| `scripts/6_VISUALIZACION.ipynb` | Visualización de datos |
| `scripts/7_ANALISIS_PREDICTIVO.ipynb` | Análisis de regresión OLS |

## Cómo ejecutar

1. Abrir `0_MASTER_SCRIPT.ipynb` en Google Colab
2. Montar Google Drive y autenticarse
3. Ejecutar todas las celdas secuencialmente
